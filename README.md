# Spark Studio

**A remix-first AI coding playground for kids.**

Spark Studio helps young creators turn ideas into playable mini-projects, inspect the code behind them, ask questions about the code, remix each other’s work, and learn the coding concepts behind every AI-assisted change.

## Core Loop

**Idea → Build → Play → Inspect Code → Ask Code → Remix → Explain → Share**

## Why

Scratch made coding creative, visual, and remixable. Spark Studio brings that same creative-learning spirit into the AI era.

The goal is to help young learners become creators, not just consumers, of AI-generated software.

## MVP Features

- Idea Wall
- Discover Projects
- Prompt-first Builder
- Playable project templates
- Project Detail page
- Play / Code / Learn tabs
- Ask the Code panel
- Remix flow
- Fork/remix lineage
- Structured reactions
- Safety checked projects

## Tech Direction

- Next.js
- React
- TypeScript
- Tailwind CSS
- Template-based project generation
- Mock data first
- Optional LLM API for generation, remixing, and code Q&A

## Safety

Spark Studio avoids open comments and private messages. Collaboration happens through remixing, structured reactions, and project lineage.

## Demo

Build “Ocean Cleanup Game,” inspect the code, ask how the score works, remix it into “Space Junk Rescue,” then show the lineage and learning summary.

## Final Line

Spark Studio turns vibe coding into guided creative learning for the next generation.
