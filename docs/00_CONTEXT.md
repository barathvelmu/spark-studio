# 00_CONTEXT.md — Product Context

## What We Are Building
**Spark Studio** is a remix-first AI coding playground for kids.

It lets young creators:
1. Start from an idea
2. Build a playable mini-project
3. Inspect the code behind it
4. Ask questions about the code
5. Remix another project
6. Preserve credit through lineage
7. Learn what changed and why

## Inspiration
Scratch made coding accessible by making it creative, visual, social, and remixable.

Spark Studio asks:
> What would Scratch feel like if it were designed for the AI-native generation?

## Social Impact
Many students have ideas but lack:
- Advanced CS classes
- Technical mentors
- Setup-heavy tools
- Confidence to start from a blank page
- AI tools designed for young learners

Spark Studio helps them become creators, not just consumers.

## Product Thesis
AI should not just generate code for kids. It should help them create, inspect, ask, remix, and understand.

## Core Loop
**Idea → Build → Play → Inspect Code → Ask Code → Remix → Explain → Share**

## Target User
Primary: ages 10–14.
Secondary: beginner teens, libraries, after-school programs, community STEM groups, mentors.

Not classroom-only.

## Final Line
Spark Studio turns vibe coding into guided creative learning for the next generation.
