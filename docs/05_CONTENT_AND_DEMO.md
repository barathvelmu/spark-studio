## Demo Path

1. Open landing page.
2. Say: “Scratch made coding creative and remixable. We asked what that looks like in the AI era.”
3. Open Idea Wall.
4. Click Ocean Cleanup Game.
5. Generate project.
6. Show Play tab.
7. Open Code tab.
8. Ask: “How does the score work?”
9. Show kid-friendly answer.
10. Click Remix.
11. Prompt: “Make it about space junk instead of ocean plastic.”
12. Show Space Junk Rescue.
13. Show “Forked from Ocean Cleanup Game.”
14. Open Learn tab.
15. Show:
    - What changed
    - What was reused
    - Concepts: variables, events, conditionals, loops
16. Publish to Discover.

## Pitch Lead (always open with this)
> Every AI-generated change becomes a learning moment.

Lead with the learning angle, not the kids angle. Anthropic judges respond to the AI/learning/safety story before the cute-kids story.

## 30-Second Pitch
Every AI-generated change becomes a learning moment. Spark Studio is a remix-first AI coding playground for kids — young creators describe an idea, turn it into a playable mini-project, inspect the real code behind it, ask questions about how it works, remix other projects, and understand the coding concepts behind every AI-assisted change. Inspired by Scratch, built for the AI-native generation, designed to reach students without mentors, advanced CS classes, or expensive tools.

## Strong Judging Lines
“Scratch lowered the barrier from syntax to blocks. Spark Studio lowers the barrier from blank page to creative software, while preserving learning through code inspection, remix explanations, and project lineage.”

“Every AI-generated change becomes a learning moment.”

“ChatGPT gives an answer. Spark Studio gives a full creative learning environment: preview, codebase, code Q&A, remix lineage, safety, and learning summaries.”

## LinkedIn Framing
Built Spark Studio at the Anthropic hackathon — an AI-native creative coding playground inspired by Scratch, designed to help young learners turn ideas into playable projects, remix each other’s work, inspect the code, and understand the concepts behind every AI-assisted change.

The bigger question we explored:
What would Scratch look like if it were designed for the AI-native generation?

## Final Line
Spark Studio turns vibe coding into guided creative learning for the next generation.
