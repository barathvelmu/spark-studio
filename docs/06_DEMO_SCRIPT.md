# Spark Studio — 2-Minute Demo Script

**Total runtime: ~2:00** · Speak at a brisk pace (~190 wpm) · Every feature appears on screen.

> **Before you hit record (do this first):**
> 1. Sign in once as your demo account — no auth modal interrupting mid-take.
> 2. Visit `/`, `/ideas`, and `/project/p_ocean` once to warm Next.js compilation.
> 3. Start recording on the landing page (`/`).
> 4. Browser zoom: 110–125% so text is readable on video.

## 0:00 – 0:12  |  Hook

**On screen:** Landing page hero — *"Make something only you would think of."*

**Say:**
> "Spark Studio. A remix-first AI coding playground for kids — where every AI-generated change becomes a learning moment. Scratch made coding creative. We brought that into the AI era. Watch."

## 0:12 – 0:25  |  Idea Wall → Build

**On screen:** Click **Ideas** in nav → click **Ocean Cleanup Game** card → builder loading dots → land on project page.

**Say:**
> "Kids hit the blank page first — so we preload kid-safe ideas. Tap Ocean Cleanup, and instead of generating risky raw code, the AI fills in a template config — turtle, plastic, ocean. Always playable, always safe."

## 0:25 – 0:40  |  Play

**On screen:** Play tab is default. Arrow keys — collect plastic 3+ times. Score ticks up on screen.

**Say:**
> "Real game, in the browser, no setup. Arrow keys move the turtle, touch the plastic, score goes up. This is what 'AI built me a project' should actually mean."

## 0:40 – 0:58  |  Code

**On screen:** Click **Code** tab → switch to **JS** file → scroll slowly past 2–3 commented blocks.

**Say:**
> "Now the difference. Hit Code — real JavaScript, plain-English comments on every block. 'Keep track of how many pieces of plastic.' The AI didn't just write code — it wrote code that teaches as you read. No black boxes."

---

## 0:58 – 1:18  |  Ask the Code

**On screen:** Click the suggested chip **"How does the score work?"** → answer renders → JS lines 1, 26, 27 highlight.

**Say:**
> "Ask the Code. Not a chatbot — it only knows about this project. 'How does the score work?' — kid-friendly answer, highlights the exact lines. Curiosity becomes learning, scoped to what they made."

## 1:18 – 1:38  |  Remix + Lineage

**On screen:** Click **🔁 Remix this** → type *"Make it about space junk instead of ocean plastic"* → hit Remix → new project page loads → hold on **"🔁 Forked from Ocean Cleanup Game"** pill.

**Say:**
> "Now I remix it. 'Make it about space junk instead of ocean plastic.' Same logic, new theme — astronaut, satellites, deep space. And look — 'Forked from Ocean Cleanup Game.' Every remix preserves credit. GitHub-style attribution, kid-friendly."

---

## 1:38 – 1:55  |  Learn + Safety

**On screen:** Click **Learn** tab → pan across What changed → Concepts chips (variables, events, conditionals, loops) → Next challenge → safety badge visible at top.

**Say:**
> "The Learn tab closes the loop. What changed, the concepts the AI used — variables, events, conditionals, loops — and a next challenge so kids keep going. Every project is safety-checked. No comments, no DMs, no personal data."

---

## 1:55 – 2:00  |  Close

**On screen:** Hold on Learn tab or cut back to hero.

**Say:**
> "ChatGPT gives an answer. Spark Studio gives a creative learning environment. Thanks."
